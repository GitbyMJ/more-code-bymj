Flow of HTML

There are several ways in which you can adjust the position of certain HTML elements using CSS! These properties are position, display, z-index, float, and clear. Lets talk about position.

Positioning like a pro!

The placement of box HTML elements can be changed with the position property. The default value is static, which pushes all elements to the left side of the page. Let's find out more about the other position properties.

Remember: Default properties and values do not need to be specified!

Relative

Use the relative property to set the position of an HTML element relative, or proportionally, to it's default position on the webpage.

This is specified with the top, bottom, left, and right offset values represented in pixels, ems, or percentages.

  • top: moves the element down from the top.
  • bottom: moves the element up from the bottom.
  • left: moves the element to the right.
  • right: moves the element to the left

Examples of the relative property at work
Example Explaination
.container {
position: relative;
top: 400px;
}		 This code will place the container element 400 pixels from the top of the webpage.

Absolute

When using the absolute property, other elements ignore the position of the element and therefore overlap it. The element will be positioned relative to its closest positioned parent element, and offset properties will be used to determine the final position. The same offset values apply here.

Example of the absolute property at work
Example Explaination
#red {
background-color: rgb(255, 0, 0);
position: absolute;
}

#blue {
background-color: rgb(0, 0, 255);
position: relative;
top: 60px;
}		 The absolute value is used to show the circle objects overlapping. This example also uses the relative value.

Fixed

Using the fixed value will fix, or "stick", the element in its set offset position regardless of when a user scrolls.

This is typically used for elements such as navigation bars!

Sticky

Using this value will cause the element to behave the same as the fixed property, but instead will stick to its position until it reaches the specified offset value, and there it will rejoin the HTML flow.
